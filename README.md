### My Setup

* Linux Distro: `arch`
* Window Manager: `openbox`
* Bar: `lemonbar`
* Launcher: `dmenu2`
* Terminal Emulator: `urxvt`
* Shell: `zsh`
* Terminal Font: `lemon`
* File Manager: `thunar`
* Text Editor: `neovim`
* Color Scheme: `ryuuko` (In this repo)
* Music Player: `mpd` + `ncmpcpp`
* Browser: `firefox`
* Startpage: In this repo under "/startpage/"
* Video Player: `mpv`
