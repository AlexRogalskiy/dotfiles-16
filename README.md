## Setup

- Application Launcher: `rofi`
- Music Player: `spotify`
- Shell: `bash`
- Terminal Emulator: `st`
- Text Editor: `nvim`
- Web Browser: `chromium`
- Window Manager: `i3`


## Installation

1. Install `stow`
2. `cd ~`
3. `git clone --depth 1 https://github.com/dylanaraps/dotfiles`
4. `git clone --depth 1 https://github.com/dylanaraps/bin`
5. `cd dotfiles`
6. `./install.sh`
