## Setup

- Application Launcher: `rofi`
- Music Player: `spotify (website)`
- Notifications: `dunst`
- Shell: `bash`
- Terminal Emulator: `urxvt`
- Text Editor: `neovim`
- Video Player: `mpv`
- Web Browser: `firefox-nightly`
- Window Manager: `openbox-patched`


## Installation

1. `cd ~`
2. `git clone --depth 1 https://github.com/dylanaraps/dotfiles`
3. `cd dotfiles`
4. `./install.sh`
