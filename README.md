# My Setup

* Linux Distro: `arch`
* Window Manager: `openbox`
* Bar: `lemonbar`
* Launcher: `dmenu2`
* Terminal Emulator: `urxvt`
* Shell: `mksh`
* Terminal Font: `lemon`
* File Manager: `thunar` and `ranger`
* Text Editor: `neovim`
* Color Scheme: `ryuuko` (In this repo)
* Music Player: `cmus`
* Browser: `firefox`
* Startpage: In this repo under "/startpage/"
* Video Player: `mpv`
